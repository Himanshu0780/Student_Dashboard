# Student Dashboard

CCE Student Dashboard is a responsive frontend web application designed
to help students manage their academic activities from a single place.
The dashboard provides features such as attendance tracking, assignment
management, timetable viewing, notices, and profile management.

This project focuses on user interface design, client-side logic, and
interactive components using vanilla web technologies.

---

## Features

- Student dashboard with sidebar navigation
- Attendance overview with percentage and subject-wise breakdown
- Assignment upload and pending assignment tracking
- Timetable display
- Notices section with downloadable files
- Profile view and edit functionality
- Light / Dark theme toggle
- Settings panel (theme, notifications, language)
- Login and logout simulation
- Responsive design for different screen sizes

---

## Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla)
- Boxicons
- Chart.js (for visual representation)

3. Use the dashboard features directly (no server required)
